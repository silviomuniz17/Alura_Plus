<h1>Lembrete de como colocar o site no ar para que possamos olhar</h1>

```
1° forma (Passo a Passo)
1 > Primeiro precisamos ir até o projeto no GitHub
2 > Já no seu projeto aperta em Settings(configurações)
3 > No menu lateral selecione Pages
4 > Na opção Branch devemos selecionar qual linha usar. Melhor a master pois é onde fica o codigo principal
5 > Logo na frente de onde selecionamos a Branch tem o botão salvar, então devemos salvar
6 > Eu apertei selecionei Pages no menu lateral novamente, depois disso apareceu o linck para acessar minhas pagina criada 
```
```
2° forma (Passo a Passo)
1 > Acessar o site https://vercel.com/new?onboarding=true
2 > Estou utilizando a mesma conta do github para me conectar na Vercel
3 > Depois que acessei minha conta tem uma opção dentro de um quadrado Import Git Repository
4 > seleciona o projeto que gostaria e importa ele 
5 > Depois no quadrado ao lado vou preencher os dados com o nome do projeto e colocar o Framework Preset
6 > Só apertar em Deploy  
```
